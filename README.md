# Guilherme Mendes - Portfolio

Personal portfolio website showcasing my work as a Backend Developer, Team Leader, and Founder.

## Live Demo

[https://guilherme-silva-mendes.github.io/portfolio/](https://guilherme-silva-mendes.github.io/portfolio/en)

## Tech Stack

- **React 18** with TypeScript
- **Vite** for build tooling
- **Tailwind CSS** for styling
- **Framer Motion** for animations
- **React Scroll** for smooth navigation
- **React Icons** for icons

## Features

- Responsive design (mobile, tablet, desktop)
- Dark theme with cyan/blue accents
- Smooth scroll navigation
- Animated sections with Framer Motion
- Bilingual support (English / Portuguese)
- Contact form

## Sections

- **Hero** - Introduction with animated taglines
- **About** - Bio and highlights (Clean Code, CI/CD, Docker, Automation, AI Agents, Backend)
- **Skills** - Tech stack with icons
- **Projects** - Featured projects with GitHub links
- **Experience** - Professional timeline
- **Contact** - Contact form and social links

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Deploy to GitHub Pages
npm run deploy
```
